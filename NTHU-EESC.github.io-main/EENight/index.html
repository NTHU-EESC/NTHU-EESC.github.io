2025 清大電機之夜

加入我們的 2025 清大電機之夜，探索電機工程的奧秘！這是一個為期一週的精彩活動，讓你親身體驗最新的電機技術，結識志同道合的朋友，並在專業導師的指導下完成令人驚嘆的項目。無論你是電機新手還是有經驗的愛好者，這裡都有適合你的精彩內容！

活動介紹

吉伴股

學習設計和製作各種有趣的電路，從簡單的 LED 閃爍器到複雜的音頻放大器，讓你的創意在電路板上綻放光芒。

場器股

深入探索 Python 和 C++ 等程式語言，學習如何控制硬體設備，開發智能系統，讓你的程式直接與現實世界互動。

人工智慧組

了解 AI 的基本原理，實際操作機器學習模型，參與激動人心的專案，如圖像識別或自然語言處理。

物聯網組

探索 IoT 的無限可能，學習如何連接和控制各種智能設備，打造屬於自己的智能家居或環境監測系統。
